## Examples

```
julia> Starbox("Hello, World!")
*******************************************************************************
*                                Hello, World!                                *
*******************************************************************************


julia> Starbox("/Hello, World!/")
*******************************************************************************
*                                                                             *
*                                Hello, World!                                *
*                                                                             *
*******************************************************************************


julia> Starbox("/Hello, World!//")
*******************************************************************************
*                                                                             *
*                                Hello, World!                                *
*                                                                             *
*                                                                             *
*******************************************************************************


julia> Starbox("*Hello, World!*", sep='*')
*******************************************************************************
*                                                                             *
*                                Hello, World!                                *
*                                                                             *
*******************************************************************************


julia> Starbox("*Hello, World!*", sep='*', star="=")
===============================================================================
=                                                                             =
=                                Hello, World!                                =
=                                                                             =
===============================================================================


julia> Starbox("*Hello, World!*", sep='*', star="+-")
+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+
+-                                                                           -+
+-                               Hello, World!                               -+
+-                                                                           -+
+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+


julia> Starbox("/Skinny box/", width=40)
****************************************
*                                      *
*              Skinny box              *
*                                      *
****************************************


julia> Starbox("/Skinny box/", width=40, star="|")
||||||||||||||||||||||||||||||||||||||||
|                                      |
|              Skinny box              |
|                                      |
||||||||||||||||||||||||||||||||||||||||


julia> Starbox("/Padded skinny box/", width=40, star="|", pad=4)
    ||||||||||||||||||||||||||||||||||||||||
    |                                      |
    |          Padded skinny box           |
    |                                      |
    ||||||||||||||||||||||||||||||||||||||||

julia> Starbox("*Hello, World!**How are you?*", sep='*', star="+-")

+-                                                                           -+
```
