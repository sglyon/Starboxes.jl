## Examples

```
julia> Starbox("Hello, World!")
*******************************************************************************
*                                Hello, World!                                *
*******************************************************************************


julia> Starbox("/Hello, World!/")
*******************************************************************************
*                                                                             *
*                                Hello, World!                                *
*                                                                             *
*******************************************************************************


julia> Starbox("/Hello, World!//")
*******************************************************************************
*                                                                             *
*                                Hello, World!                                *
*                                                                             *
*                                                                             *
*******************************************************************************


julia> Starbox("*Hello, World!*", '*')
*******************************************************************************
*                                                                             *
*                                Hello, World!                                *
*                                                                             *
*******************************************************************************


julia> Starbox("*Hello, World!*", '*', "=")
===============================================================================
=                                                                             =
=                                Hello, World!                                =
=                                                                             =
===============================================================================


julia> Starbox("*Hello, World!*", '*', "+-")
```
